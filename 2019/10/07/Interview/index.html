2、数组乱序

数组乱序故名思议就是把一个数组打乱。

function shuffle(arr) {
      for (let i=arr.length-1; i>=0; i--) {
          let rIndex = Math.floor(Math.random()*(i+1));
          // 打印交换值
          // console.log(i, rIndex);
          let temp = arr[rIndex];
          arr[rIndex] = arr[i];
          arr[i] = temp;
      }
      return arr;
  }

3、for in 和 for of 区别

1、在遍历对象的时候最好用for in 遍历数组的时候用for of
2、for in 循环出的是key for of循环出的是value
3、⚠️for of 是ES6新引入的属性，弥补了for in 的不足
4、for of 不能循环普通对象，需要和Object.keys()搭配使用

4、Promise接收的函数中resolve()后的代码是否会执行？

我们看一个列子

var promise = new Promise(function(resolve, reject){
    setTimeout(function(){
        resolve(1);
        console.log(2);
    }, 1000);
    console.log(3);
});
promise.then(function(data){
    console.log(data);
});

你觉得它会返回什么呢？

3
2
1

因为resolve相当于一个异步调用，属于异步触发，即状态变化后不会马上执行事件的响应函数，而是等到当前任务队列为空的时候才会执行，所以2会在1之前打印出来.
综合以上Promise构造函数接收的函数中，调用resolve()之后的代码仍然会执行，它在被new的时候是同步执行的。

6、图片懒加载和预加载

懒加载
懒加载就是先将页面内的所有图片的真实地址放在一个属性(data-url)里, 并将src先设置为1px的小图片，当触发某些条件(图片进入视野)时，再将图片的src换成真实地址。使页面加载速度快、可以减轻服务器的压力，节约了流量，用户体验好。

预加载
提前加载图片，当用户需要查看时可直接从本地缓存中渲染，可以使用css或js实现。

如果想了解更多，可以看看这篇文章。如何处理图片懒加载和预加载
